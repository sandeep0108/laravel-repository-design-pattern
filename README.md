# Installation Instructions
<ul>
<li>
git clone https://github.com/sandeep0108/laravel-repository-design-pattern.git
</li>
<li>
cd laravel-repository-design-pattern
</li>
<li>
composer install
</li>
<li>
php artisan serve
</li>
</ul>
